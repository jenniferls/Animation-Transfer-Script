Animation transfer script

This script can be used to transfer animations from one skeleton to another regardless of joint orientation differences.

------------------------------------------
		How to use:
------------------------------------------

Make sure all of the included files are placed together in a folder.
Append the path by modifying "path" in AnimationTransferMain.py.
To run the script, simply run AnimationTransferMain.py via the Maya script editor.
